> A premium, elegant and minimal portfolio website for **Areeba Mustafa**, Full Stack Developer.

Built with a soft feminine editorial aesthetic inspired by Apple, Framer, Linear and luxury boutique design. Focused on **whitespace, typography and the projects themselves** — not busy dashboards or corporate templates.


---

## ✨ Features

- **Elegant hero** with large serif typography and a soft lavender halo portrait
- **Featured projects** as horizontal editorial panels — number, screenshot, tags, one-click Live Demo
- **About section** with education & location cards
- **Tech Stack** grid using **real official brand logos** (React, Next.js, Node.js, Tailwind, TypeScript, Python, PostgreSQL, MongoDB, Git, GitHub, VS Code, Postman & more)
- **Contact** section with email, LinkedIn, GitHub & location — no phone, no form
- **Premium footer** with quick links, socials & copyright
- **Scroll-based active nav highlighting**, sticky glass navbar, smooth scroll, back-to-top button
- **Fully responsive** across desktop, laptop, tablet and mobile
- **Custom favicon** and matching AM monogram branding

---

## 🎨 Design System

| Token          | Value        |
| -------------- | ------------ |
| Background     | `#FFF9FC`    |
| Cards          | `#FFFFFF`    |
| Primary Purple | `#8B5CF6`    |
| Soft Pink      | `#F9A8D4`    |
| Lavender       | `#F3E8FF`    |
| Light Purple   | `#E9D5FF`    |
| Text           | `#18181B`    |
| Text Secondary | `#71717A`    |
| Border         | `#ECECEC`    |

**Typography**
- Headings: **Cormorant Garamond** (serif, italic accents)
- Body: **Inter** (sans-serif)

**UI Rules**
- Radius `24–32px`
- Very soft, almost invisible shadows
- Generous whitespace — 120–160px section padding
- No dashboard cards, no counters, no timelines, no glassmorphism spam

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
